# LoceriaTrujillo CRM

Sistema de Gestión de Clientes para una distribuidora de locería.

## 🚀 Estructura del Proyecto

- `frontend/`: Aplicación Next.js (App Router, TypeScript, TailwindCSS).
- `backend/`: API REST en Laravel.

## 🛠️ Requisitos

- Node.js 18+
- PHP 8.2+ (o Docker para el backend)
- PostgreSQL

## 🐳 Docker (Recomendado para Pruebas/SaaS)

El proyecto está listo para ser desplegado con Docker:

1. Asegúrate de tener Docker y Docker Compose instalados.
2. Ejecuta: `docker compose up -d`
3. El sistema estará disponible en:
   - Frontend: `http://localhost:3000`
   - Backend: `http://localhost:8000`
